# News App

- I have developed a news mobile app.

**Tools that I used** :

- Navigation Component
- Hilt for dependency injection
- Retrofit 
- Postman for Api testing.
- Coroutines 
- Livedata.
- View Binding
- Glide for image loading
- RESTApi


**Arhictecture:**

- MVVM
